# RabbitMQ

## Requirements
* Erlang
* RabbitMQ
* Node 12+

## Installing

1. `$ npm i`
2. `$ cp .env.example .env`
3. Edit `.env` file
4. `$ npm start`
5. Start each service in a new terminal :arrow_down:
6. `$ npm start:market`
7. `$ npm start:items`
8. `$ npm start:users`
9. `$ npm start:notifications`
10. Register users and create item to test
